# vue3-taste-js

JavaScript版本的Vue 3.0（极简）尝鲜版。
只是单纯为了试验新的API。

具体功能是这样的，做了一个简单的todo list，可以增删，可以搜索。
修改因为要写对话框，不想写，就没写。

启动之前先去server目录下启动server，因为需要测试异步。
